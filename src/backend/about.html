About Us

Welcome to our Flipkart Clone website. This project is made by our team for learning web development using HTML and CSS. Our aim is to create a simple e-commerce website interface similar to Flipkart.

This website contains multiple pages like Home, Login, Register, About Us and Contact Us pages connected together using navigation links. We have also used product cards, categories and responsive layouts without using java script in this project.

This project helped us understand frontend development concepts such as forms, flexbox, navigation bars, cards, hover effects and page linking.
